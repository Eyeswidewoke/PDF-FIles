# Legal Notes

## Source basis

This repository is an indexing and research interface built from publicly released materials and derived processing outputs.

## Interpretation boundary

- Mentions are not allegations.
- Associations in source text are not legal findings.
- This project does not adjudicate claims.

## Redactions and sensitive content

- Preserve official redactions where present.
- Do not publish newly identifying details for protected individuals.
- If accidental exposure is identified, report through `SECURITY.md`.

## Purpose

Primary purpose is traceability:

- claim -> cited source artifact
- source preservation and reviewability
- reproducible navigation and indexing
