## EVIDENCE TIERS

|------|------------|----------|
| **Tier 1** | Primary sources — official documents, sworn testimony, direct recordings | Mueller Report, court filings, video |

## GRADING KEY

| Grade | Meaning |
|-------|---------|
| **A** | Definitive — overwhelming evidence |
| **B** | Strong — substantial evidence, minor gaps |
| **C** | Moderate — credible but uncertainties |
| **D** | Weak — some support, major gaps |
| **F** | Debunked — contradicted by evidence |

| Status | Meaning |
|--------|---------|
| **Established** | Proven; would hold in court |
| **Highly Likely** | Strong circumstantial; expert consensus |
| **Plausible** | Supported but not conclusive |
| **Unclear** | Genuinely uncertain |
| **Likely False** | Evidence leans against |
| **False** | Definitively debunked |
