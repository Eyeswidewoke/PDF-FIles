What We Found

The Entire Library Is Gone

This is not a case of selective removal. The DOJ has taken down every page, every download, and every individual PDF associated with the Epstein Files Transparency Act release. The main landing page returns a 404 error. Every file URL redirects to a generic age-verification page.

Deceptive HTTP Responses

The dataset ZIP URLs (justice.gov/epstein/files/DataSet N.zip) still return HTTP status code 200, which normally means “OK — file available.” But the actual content served is an HTML page, not a ZIP file. Any automated monitoring tool checking only status codes would incorrectly report these files as still available. We verified this by checking the actual bytes returned — the first bytes are 0a3c2144 (HTML), not 504b0304 (ZIP magic number).

How We Verified This

  1. Local inventory: Counted all 900,956 PDF files across our complete copies of DataSets 1–12
  2. Wayback Machine CDX API: Queried Internet Archive for every PDF ever archived under justice.gov/multimedia/Court Records/ and justice.gov/multimedia/DOJ Disclosures/
  3. Live HTTP verification: Sent HEAD requests to 25 randomly-selected URLs. Checked Content-Type headers and actual bytes returned.
  4. ZIP integrity check: Downloaded first 512 bytes of dataset ZIP URLs. Received HTML (0a3c2144) instead of ZIP magic bytes (504b0304).

This data is preserved.
