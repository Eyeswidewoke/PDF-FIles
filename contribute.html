📦 HOW THIS PROJECT HANDOFFS

This repo is a base archive and index intended to be reused, merged, and extended by others. The long-term goal is not to centralize everything here.

🎯 Project intent

The goal is to create a single usable reference layer for the released files and associated research artifacts: searchable structure, source-linked records, and consistent file indexing.

This repository is the starting base for that effort. The expected path is that independent researchers and other projects use this material in their own systems, and eventually merge overlapping work into broader combined archives.

In short: publish the foundation clearly, make it easy to reuse, then step back.

🗃 What to do with this repo

  • Use the indexed pages, cast profiles, timelines, and dataset file inventories as a starting dataset.
  • Fork or mirror it into your own stack (static, database, notebook, newsroom workflow, etc.).
  • Merge selected parts into other evidence or OSINT projects where they fit.
  • Preserve source traceability: keep DocID/page references attached to claims.

📝 Corrections are still useful

This is not a call for a managed contributor program. But if you see a factual error, broken path, or source mismatch, a correction report is still valuable.

  • Best format: claim + exact file reference (DocID/page/path) + suggested correction.
  • Keep notes source-linked and specific.
